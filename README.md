# pptx-creator

Creates modern, well-structured PPTX presentations from a simple JSON data file. This skill uses a layout-driven approach for reliability and speed, and includes a mandatory interactive theme selection to ensure visual consistency.

## Key Features

*   根据用户提供的原始内容（教材章节、文章、笔记），生成 `presentation.json`。
*   通过构建脚本生成 `.pptx` 文件。

## Installation

To install `pptx-creator`, simply copy the project files to your `~/.gemini/skills/` directory.

## Usage

After installation, you can use `pptx-creator` by providing a prompt that references a document to create a slideshow.

**Example:**
```
依据 示例文档.md , 创建幻灯片。
```
